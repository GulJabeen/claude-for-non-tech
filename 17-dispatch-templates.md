# Claude for Non-Tech — Episode 17: Dispatch Templates
**Series:** Claude for Non-Tech by @techwithgul | **Keyword:** DISPATCH

---

## What is Dispatch?

Dispatch is Claude's scheduled task feature — you write a task once, set a schedule, and Claude runs it automatically at the time you choose, every time.

---

## How to set up a Dispatch task

1. Open Claude Desktop and go to **Dispatch** in the left sidebar
2. Click **New Task** and give it a name (e.g. "Weekly AI News Summary")
3. Paste your task instructions in the box (use one of the templates below)
4. Set the schedule — choose the day, time, and how often it repeats
5. Click **Save** — Dispatch will run automatically from now on

---

## 3 ready-to-use Dispatch templates

---

### Template 1: Weekly AI News Summary (every Monday at 8am)

Copy this into the task box:

```
Task: Search for the top 5 AI news stories from the past 7 days.
Write a 5-bullet summary. Keep each bullet under 20 words.
Save to: [Notion page / folder name]
Schedule: Every Monday at 8:00am
```

**Filled-in version:**
```
Task: Search for the top 5 AI news stories from the past 7 days.
Write a 5-bullet summary. Keep each bullet under 20 words.
Include: story name, what happened, why it matters.
Save to: my Notion page called "Weekly AI Digest"
Schedule: Every Monday at 8:00am
```

---

### Template 2: Daily Task Brief (every morning)

Copy this into the task box:

```
Task: Check my [calendar/task list]. List my top 3 priorities for today.
Add one sentence of context for each.
Format: numbered list
Schedule: Every weekday at 7:30am
```

**Filled-in version:**
```
Task: Check my Google Calendar for today. List my top 3 priorities — 
combining meetings, deadlines, and any flagged tasks.
For each priority, add one sentence explaining why it matters today.
Format: numbered list, plain text
Schedule: Every weekday at 7:30am
```

---

### Template 3: Monthly Business Report

Copy this into the task box:

```
Task: Pull data from [tool]. Write a monthly summary covering:
1. What went well
2. What needs attention
3. One recommendation for next month
Save to: [destination]
Schedule: First day of every month at 9:00am
```

**Filled-in version:**
```
Task: Pull last month's data from my Google Sheets sales tracker.
Write a monthly summary in 3 sections:
1. What went well — highlight the top-performing product and revenue total
2. What needs attention — flag any product with 0 or declining sales
3. One recommendation for next month — based on the data, suggest one action
Save to: paste as a new page in Notion called "Monthly Report [Month]"
Schedule: First day of every month at 9:00am
```
