## Recommended learning path for non-tech people

### Stage 1: Get comfortable (Weeks 1–2)
- Install Claude Desktop and create your first Project (→ `01-claude-desktop-setup.md`)
- Use the 3-line brief template for every task (`02-mindset-brief-template.md`)
- Learn the 3-part prompt formula and try it for 5 different tasks (`03-prompt-formula-template.md`)
- Goal: Claude feels like a useful tool you reach for naturally

### Stage 2: Build your setup (Weeks 3–4)
- Write your Project instructions so Claude knows your context (`04-project-setup-template.md`)
- Install 2–3 plugins that connect to apps you use daily (`06-plugin-install-guide.md`)
- Write your first Skill for your most repeated task (`08-skill-template.md`)
- Goal: Claude knows who you are, has access to your tools, and has a consistent voice

### Stage 3: Run full workflows (Month 2+)
- Write your first agent instruction for a multi-step task (`12-build-your-first-agent.md`)
- Combine your Skill + Plugin + Agent into one trigger prompt (`13-full-workflow-setup.md`)
- Try Dispatch to automate recurring tasks on a schedule (`17-dispatch-templates.md`)
- Goal: Claude handles whole workflows while you focus on decisions, not repetitive tasks
