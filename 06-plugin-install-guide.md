## 5-step install guide

1. Open **Claude Desktop** and click the plug icon or go to **Settings → Integrations**
2. Click **Browse Plugins** — this opens the plugin directory at https://claude.com/plugins
3. Find the plugin you want (e.g. Gmail) and click **Install** or **Connect**
4. You'll be asked to sign in to the app (e.g. sign in to your Google account) — this gives Claude permission to access it
5. Go back to Claude Desktop, start a new chat, and test it by asking Claude to do something with that app

That's it. No code. No technical setup.

## Troubleshooting: 3 common issues + fixes

**Issue 1: "Claude doesn't seem to be using the plugin"**
- Fix: Start a fresh chat. Plugins sometimes don't activate in an existing conversation. Type: "Use Gmail to check my inbox" to make it explicit.

**Issue 2: "I connected the plugin but Claude says it can't access my files/emails"**
- Fix: Go back to Settings → Integrations, disconnect the plugin, and reconnect. Make sure you're signing in with the correct account (e.g. the right Google account if you have more than one).

**Issue 3: "The plugin installed but I can't find it"**
- Fix: Close and reopen Claude Desktop. New plugins sometimes need a restart to appear. If it still doesn't show, check Settings → Integrations to confirm it's listed there.
