## The workflow prompt formula

A workflow prompt tells Claude to complete a task in a specific sequence of steps — like a recipe it follows from start to finish. Copy this, fill it in, and paste into Claude.

```
Goal: [what you want done at the end]
Steps:
1. [first action]
2. [second action]
3. [third action]
Save to: [where the result should go]
```

---

## 3 filled-in workflow examples

**Content creator — weekly post planning**
```
Goal: A ready-to-post Instagram caption with hashtags for a reel about morning routines
Steps:
1. Write a hook (first line) that stops the scroll — bold claim or surprising fact
2. Write 3–4 short body lines explaining the routine (each under 10 words)
3. End with a question to boost comments, then list 10 relevant hashtags
Save to: paste the result directly in this chat so I can copy it
```

**Business owner — weekly email newsletter**
```
Goal: A 300-word email newsletter for my candle brand subscribers
Steps:
1. Write a subject line (max 8 words, no emojis)
2. Write a short personal opening (2 sentences — what's new this week)
3. Highlight one product with a 2-sentence description and a CTA button (text only)
4. Close with a warm sign-off from me, "Layla"
Save to: paste as plain text so I can copy it into Mailchimp
```

**Student — essay research summary**
```
Goal: A structured summary I can use to start writing my essay introduction
Steps:
1. List the 5 most important points from the text I paste below
2. For each point, write one sentence explaining why it matters to my argument
3. Suggest one counterargument I should address
4. Write a 3-sentence introduction I can use as a starting draft
Save to: paste everything in this chat with clear headings
```

---

## Tips for better workflow prompts

- **Be specific about the output format.** Say "numbered list," "plain text," "table," or "bullet points" — Claude will match it exactly.
- **Use "Save to" to tell Claude where the result should land.** Even if it's just "paste here," it helps Claude know it's done.
- **Add one example if you have one.** "Write a hook like this: [example]" cuts revision time in half.
